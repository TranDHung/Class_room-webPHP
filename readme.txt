Cách setup để chạy được website:
- Copy tất cả nội dung trong thư mục source code, và paste vào thư mục htdocs trong xampp.
- Mở xampp control, start module Apache và MySQL sau đó mở trình duyệt lên truy cập localhost:8888/phpmyadmin, import database.sql
- Sau đó truy cập localhost:8888 để vào website.
